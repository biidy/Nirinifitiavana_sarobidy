## 🚀 Projets À la Une

Voici un aperçu des projets majeurs que j'ai conçus et déployés :

### 🌾 1. Système de Recommandation de Cultures (AgriTech)
Un modèle prédictif basé sur le Machine Learning qui recommande les cultures optimales en fonction des caractéristiques du sol et des conditions environnementales.
* **Tech Stack :** Python, XGBoost, FastAPI, Streamlit.
* **Impact :** Optimisation des rendements agricoles grâce à une approche basée sur les données.
* [🔗 Voir le code source](https://github.com/votre-utilisateur/crop-recommendation) | [🌐 Tester la démo en direct](https://huggingface.co/spaces/votre-utilisateur/crop-prediction)

Un outil d'aide à la rédaction basé sur le Traitement Automatique du Langage Naturel (NLP) intégrant de l'autocorrection, de la lemmatisation et de la prédiction du prochain mot.
* **Tech Stack :** Python, Hugging Face Transformers (GPT-2), FastAPI, Next.js.
* [🔗 Voir le code source](https://github.com/votre-utilisateur/intelligent-writing-assistant)

### 🤖 3. Pipeline MLOps & CI/CD Automatisé
Mise en place d'une infrastructure complète pour automatiser l'entraînement, le test et le déploiement continu d'un modèle de prédiction d'annonces publicitaires.
* **Tech Stack :** GitHub Actions, Docker, MLRun, Hugging Face Spaces.
